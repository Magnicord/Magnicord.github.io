Python Basics Data Types: Lists, Tuples and Sets

This note is compiled based on Corey Schafer’s Python Tutorial.
And it mainly focuses on summarizing knowledge.

Lists

List is a collection which is:

  • ordered
  • changeable

Creating Lists

We use square bracket notation to represent a list.

Similar to string, we can use len to get the length of the list.

Accessing Lists

There are two ways to access lists:

  • Indexing
  • Slicing

Let ls be a list containing mmm elements.

Indexing

Operation Description
ls[i] Access the i+1thi+1^{th}i+1th element.
ls[-i] Access the iii to last element.

Slicing

Operation Description
ls[i:j] Access elements in the location [i,j)[i, j)[i,j).
ls[:j] Access elements in the location [0,j)[0, j)[0,j).
ls[i:] Access elements in the location [i,m][i, m][i,m].
ls[:] Access elements in the location [0,m][0, m][0,m].

remove Remove a specific element.

Sorting Lists

Using sort Method

Using sorted Function

Tuples

Tuple is a collection which is:

  • ordered
  • unchangeable

Creating Tuples

We use parenthesis notation to represent a tuple.

Mutable and Immutable

Lists and tuples have most methods in common.

The significant difference between list and tuple is mutability.

  • list: Mutable
  • tuple: Immutable

We use an example of list and tuple modifying elements to interpret mutability.

Sets

Set is a collection which is:

  • unordered
  • unchangeable

Especially, sets allow no duplicates.

Creating Sets

We use curly bracket notation to represent a tuple.

The result of print changes with every execution due to sets’ unordered feature.

Allowing No Duplicates

Sets Operation

Operation Notation Description
intersection A∩BA\cap BA∩B Find elements in both set AAA and BBB.
union A∪BA\cup BA∪B Combine set AAA and BBB.
difference A∖BA\setminus BA∖B Find elements in set AAA and not in set BBB.

Creating Empty Lists, Tuples and Sets

We can use constructor to create lists, tuples and sets.

For sets, we can only use constructor (i.e. set()) to create an empty one.
